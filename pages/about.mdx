# About

### 안녕하세요, 장태영입니다.

2년차 프론트엔드 개발자로 React, React Native 등의 기술을 사용하여 프로덕션 서비스를 개발 및 운영한 경험이 있습니다.

포스팅 수집을 좋아해서 개인 시간에 라이브러리 Maintainer들의 블로그로 여행을 떠나며 이때 발견한 좋은 이론이나 예제를 사용해보는 것을 즐기고 있습니다.

`모든 것은 trade-off다`라는 말을 좋아합니다. 원하는 바를 조건 없이 이뤄주는 산타 같은 존재는 없고 내가 활용할 수 있는 자원은 한정되어 있기 때문에 발생하는 요구사항에 맞춰 최대한의 결과물을 만들어 낼 수 있도록 노력합니다.

최근에는 Bottom up 방식의 컴포넌트 설계에 관심이 있고 TDD를 맛보고 있습니다.
